API version 2025.0

The basic representation of a Box Doc Gen batch object. A Box Doc Gen batch contains one or more Box Doc Gen jobs.

* [Base](/reference/v2025.0/resources/docgen-batch--base/)

query parameter, additional to the fields specified in the
parameter.
