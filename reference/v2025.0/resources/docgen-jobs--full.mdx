---
title: "Box Doc Gen jobs (Full)"
---

API version 2025.0

A list of Box Doc Gen jobs with a full set of parameters.

* [Full](/reference/v2025.0/resources/docgen-jobs--full/)
* [Standard](/reference/v2025.0/resources/docgen-jobs/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **full** variant can be returned
by API endpoints that support the `fields` parameter. For example,
by defining the `fields` request parameter as
`id,type` when requesting a file by ID, only those fields will
be returned in the API response.
</Info>

<ParamField body="entries" type="Box Doc Gen job (Full)array">
List of jobs.
</ParamField>

<ParamField body="limit" type="integer(int64)">
example: `1000`

The limit that was used for these entries. This will be the same as the
`limit` query parameter unless that value exceeded the maximum value
allowed. The maximum value varies by API.
</ParamField>

<ParamField body="next_marker" type="string">
example: `JV9IRGZmieiBasejOG9yDCRNgd2ymoZIbjsxbJMjIs3kioVii`

The marker for the start of the next page of results.
</ParamField>

<ParamField body="prev_marker" type="string">
example: `JV9IRGZmieiBasejOG9yDCRNgd2ymoZIbjsxbJMjIs3kioVih`

The marker for the start of the previous page of results.
</ParamField>
