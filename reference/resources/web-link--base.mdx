Web links are objects that point to URLs. These objects
are also known as bookmarks within the Box web application.

Web link objects are treated similarly to file objects,
they will also support most actions that apply to regular files.

* [Standard](/reference/resources/web-link/)
* [Mini](/reference/resources/web-link--mini/)
* [Base](/reference/resources/web-link--base/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **base** variant represent
the default fields that are always returned when using the `fields`
query parameter, additional to the fields specified in the
parameter.
</Info>

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for this web link.
</ParamField>

<ParamField body="type" type="string">
example: `web_link`

The value will always be `web_link`.

Value is always `web_link`
</ParamField>

<ParamField body="etag" type="string">
example: `1`

The entity tag of this web link. Used with `If-Match`
headers.
</ParamField>
