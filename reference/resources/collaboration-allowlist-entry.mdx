An entry that describes an approved domain for which users can collaborate
with files and folders in your enterprise or vice versa.

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for this entry.
</ParamField>

<ParamField body="type" type="string">
example: `collaboration_whitelist_entry`

The value will always be `collaboration_whitelist_entry`.

Value is always `collaboration_whitelist_entry`
</ParamField>

<ParamField body="created_at" type="string(date-time)">
example: `2012-12-12T10:53:43-08:00`

The time the entry was created at.
</ParamField>

<ParamField body="direction" type="string">
example: `both`

The direction of the collaborations to allow.

Value is one of `inbound`,`outbound`,`both`
</ParamField>

<ParamField body="domain" type="string">
example: `example.com`

The whitelisted domain.
</ParamField>

<ParamField body="enterprise" type="object">
The enterprise this list is applied to.

  </ParamField>

  The name of the enterprise.
  </ParamField>

</Expandable>
</ParamField>
