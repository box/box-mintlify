API version 2024.0

<Note>
This resource is used by endpoints in the version **2024.0**.
For more details, see **[Box API versioning](/guides/api-calls/api-versioning-strategy/)**. 

Learn more about [Box SDK versioning strategy](/guides/tooling/sdks/sdk-versioning).
</Note>

An instance of a metadata template, which has been applied to a file or
folder.

* [Full](/reference/resources/metadata--full/)
* [Standard](/reference/resources/metadata/)
* [Base](/reference/resources/metadata--base/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **full** variant can be returned
by API endpoints that support the `fields` parameter. For example,
by defining the `fields` request parameter as
`id,type` when requesting a file by ID, only those fields will
be returned in the API response.
</Info>

<ParamField body="$canEdit" type="boolean">
example: `true`

Whether the user can edit this metadata instance.
</ParamField>

<ParamField body="$id" type="string(uuid)">
example: `01234500-12f1-1234-aa12-b1d234cb567e`

A UUID to identify the metadata instance.
</ParamField>

<ParamField body="$parent" type="string">
example: `folder_59449484661,`

The identifier of the item that this metadata instance
has been attached to. This combines the `type` and the `id`
of the parent in the form `{type}_{id}`.
</ParamField>

<ParamField body="$scope" type="string">
example: `enterprise_27335`

An ID for the scope in which this template
has been applied. This will be `enterprise_{enterprise_id}` for templates
defined for use in this enterprise, and `global` for general templates
that are available to all enterprises using Box.
</ParamField>

<ParamField body="$template" type="string">
example: `marketingCollateral`

The name of the template.
</ParamField>

<ParamField body="$type" type="string">
example: `properties-6bcba49f-ca6d-4d2a-a758-57fe6edf44d0`

A unique identifier for the "type" of this instance. This is an
internal system property and should not be used by a client
application.
</ParamField>

<ParamField body="$typeVersion" type="integer">
example: `2`

The last-known version of the template of the object. This is an
internal system property and should not be used by a client
application.
</ParamField>

<ParamField body="$version" type="integer">
example: `1`

The version of the metadata instance. This version starts at 0 and
increases every time a user-defined property is modified.
</ParamField>
