File version legal hold is an entity representing all
holds on a File Version.

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for this file version legal hold.
</ParamField>

<ParamField body="type" type="string">
example: `file_version_legal_hold`

The value will always be `file_version_legal_hold`.

Value is always `file_version_legal_hold`
</ParamField>

<ParamField body="deleted_at" type="string(date-time)">
example: `2012-12-12T10:53:43-08:00`

Time that this File-Version-Legal-Hold was
deleted.
</ParamField>

<ParamField body="file" type="File (Mini)object">
The file for the file version held.
Note that there is no guarantee that the current
version of this file is held.
</ParamField>

<ParamField body="file_version" type="File version (Mini)object">
The file version that is held.
</ParamField>

<ParamField body="legal_hold_policy_assignments" type="Legal hold policy assignmentarray">
List of assignments contributing to this Hold.
</ParamField>
