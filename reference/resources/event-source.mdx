---
title: "Event source"
---

API version 2024.0



The source file or folder that triggered an event in
the event stream.

<ParamField body="classification" type="object">
The object containing classification information for the item that
triggered the event. This field will not appear if the item does not
have a classification set.

<Expandable>
  <ParamField body="name" type="string">
  example: `Top Secret`
  
  The classification's name.
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="item_id" type="string">
example: `560284318361`

The unique identifier that represents the
item.
</ParamField>

<ParamField body="item_name" type="string">
example: `report.pdf`

The name of the item.
</ParamField>

<ParamField body="item_type" type="string">
example: `file`

The type of the item that the event
represents. Can be `file` or `folder`.

Value is one of `file`,`folder`
</ParamField>

<ParamField body="owned_by" type="User (Mini)object">
The user who owns this item.
</ParamField>

<ParamField body="parent" type="Folder (Mini)object">
The optional folder that this folder is located within.

This value may be `null` for some folders such as the
root folder or the trash folder.
</ParamField>
