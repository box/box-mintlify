---
title: "User"
---

API version 2024.0



A standard representation of a user, as returned from any
user API endpoints by default.

* [Full](/reference/resources/user--full/)
* [Standard](/reference/resources/user/)
* [Mini](/reference/resources/user--mini/)
* [Base](/reference/resources/user--base/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **standard** variant are generally
returned when the resource is requested via its own API endpoints.
For example, when retrieving a file by ID, it will return
these fields unless the `fields` parameter has been specified.
</Info>

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for this user.
</ParamField>

<ParamField body="type" type="string">
example: `user`

The value will always be `user`.

Value is always `user`
</ParamField>

<ParamField body="address" type="string">
example: `900 Jefferson Ave, Redwood City, CA 94063`

The user’s address.
</ParamField>

<ParamField body="avatar_url" type="string">
example: `https://www.box.com/api/avatar/large/181216415`

URL of the user’s avatar image.
</ParamField>

<ParamField body="created_at" type="string(date-time)">
example: `2012-12-12T10:53:43-08:00`

When the user object was created.
</ParamField>

<ParamField body="job_title" type="string">
example: `CEO`

The user’s job title.
</ParamField>

<ParamField body="language" type="string">
example: `en`

The language of the user, formatted in modified version of the
[ISO 639-1](/guides/api-calls/language-codes/) format.
</ParamField>

<ParamField body="login" type="string(email)">
example: `ceo@example.com`

The primary email address of this user.
</ParamField>

<ParamField body="max_upload_size" type="integer(int64)">
example: `2147483648`

The maximum individual file size in bytes the user can have.
</ParamField>

<ParamField body="modified_at" type="string(date-time)">
example: `2012-12-12T10:53:43-08:00`

When the user object was last modified.
</ParamField>

<ParamField body="name" type="string">
example: `Aaron Levie`

The display name of this user.
</ParamField>

<ParamField body="notification_email" type="object">
An alternate notification email address to which email
notifications are sent. When it's confirmed, this will be
the email address to which notifications are sent instead of
to the primary email address.

<Expandable>
  <ParamField body="email" type="string">
  example: `notifications@example.com`
  
  The email address to send the notifications to.
  </ParamField>

  <ParamField body="is_confirmed" type="boolean">
  example: `true`
  
  Specifies if this email address has been confirmed.
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="phone" type="string">
example: `6509241374`

The user’s phone number.
</ParamField>

<ParamField body="space_amount" type="integer(int64)">
example: `11345156112`

The user’s total available space amount in bytes.
</ParamField>

<ParamField body="space_used" type="integer(int64)">
example: `1237009912`

The amount of space in use by the user.
</ParamField>

<ParamField body="status" type="string">
example: `active`

The user's account status.

Value is one of `active`,`inactive`,`cannot_delete_edit`,`cannot_delete_edit_upload`
</ParamField>

<ParamField body="timezone" type="string(timezone)">
example: `Africa/Bujumbura`

The user's timezone.
</ParamField>
