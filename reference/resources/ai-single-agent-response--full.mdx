---
title: "AI agent (Full)"
---

API version 2024.0



Full representation of an AI Agent instance.

* [Full](/reference/resources/ai-single-agent-response--full/)
* [Standard](/reference/resources/ai-single-agent-response/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **full** variant can be returned
by API endpoints that support the `fields` parameter. For example,
by defining the `fields` request parameter as
`id,type` when requesting a file by ID, only those fields will
be returned in the API response.
</Info>

<ParamField body="id" type="string">
example: `1234567890`

The unique identifier of the AI Agent.
</ParamField>

<ParamField body="type" type="string">
example: `ai_agent`

The type of agent used to handle queries.

Value is always `ai_agent`
</ParamField>

<ParamField body="access_state" type="string">
example: `enabled`

The state of the AI Agent. Possible values are: `enabled`, `disabled`, and `enabled_for_selected_users`.
</ParamField>

<ParamField body="allowed_entities" type="The entity with type and ID[Group (Base)/User (Base)]array">
List of allowed users or groups.
</ParamField>

<ParamField body="ask" type="AI agent capability askobject">
The AI agent to be used to ask questions.
</ParamField>

<ParamField body="created_at" type="string(date-time)">
example: `2022-01-01T00:00:00Z`

The ISO date-time formatted timestamp of when this AI agent was created.
</ParamField>

<ParamField body="created_by" type="User (Base)object">
The user who created this agent.
</ParamField>

<ParamField body="extract" type="AI agent capability extractobject">
The AI agent to be used for metadata extraction.
</ParamField>

<ParamField body="icon_reference" type="string">
example: `https://cdn01.boxcdn.net/app-assets/aistudio/avatars/logo_analytics.svg`

The icon reference of the AI Agent.
</ParamField>

<ParamField body="modified_at" type="string(date-time)">
example: `2022-01-01T00:00:00Z`

The ISO date-time formatted timestamp of when this AI agent was recently modified.
</ParamField>

<ParamField body="modified_by" type="User (Base)object">
The user who most recently modified this agent.
</ParamField>

<ParamField body="name" type="string">
example: `This is my Agent`

The name of the AI Agent.
</ParamField>

<ParamField body="origin" type="string">
example: `CUSTOM`

The provider of the AI Agent.
</ParamField>

<ParamField body="text_gen" type="AI agent capability text generationobject">
The AI agent to be used to generate text.
</ParamField>
