Represents a configured webhook.

* [Standard](/reference/resources/webhook/)
* [Mini](/reference/resources/webhook--mini/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **mini** variant are generally
returned when the resource is nested within the response for
another resource.
</Info>

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for this webhook.
</ParamField>

<ParamField body="type" type="string">
example: `webhook`

The value will always be `webhook`.

Value is always `webhook`
</ParamField>

<ParamField body="target" type="object">
The item that will trigger the webhook.

<Expandable>
  <ParamField body="id" type="string">
  example: `1231232`
  
  The ID of the item to trigger a webhook.
  </ParamField>

  <ParamField body="type" type="string">
  example: `file`
  
  The type of item to trigger a webhook.
  
  Value is one of `file`,`folder`
  </ParamField>

</Expandable>
</ParamField>
