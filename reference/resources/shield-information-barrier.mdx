A standard representation of a
shield information barrier object.

* [Standard](/reference/resources/shield-information-barrier/)
* [Base](/reference/resources/shield-information-barrier--base/)

<Info>
This resource has a few variations that can be encountered when
using the API.

The fields that are part of the **standard** variant are generally
returned when the resource is requested via its own API endpoints.
For example, when retrieving a file by ID, it will return
these fields unless the `fields` parameter has been specified.
</Info>

<ParamField body="id" type="string">
example: `11446498`

The unique identifier for the shield information barrier.
</ParamField>

<ParamField body="type" type="string">
example: `shield_information_barrier`

The type of the shield information barrier.

Value is always `shield_information_barrier`
</ParamField>

<ParamField body="created_at" type="string(date-time)">
example: `2020-06-26T18:44:45.869Z`

ISO date time string when this
shield information barrier object was created.
</ParamField>

<ParamField body="created_by" type="User (Base)object">
The user who created this shield information barrier.
</ParamField>

<ParamField body="enabled_at" type="string(date-time)">
example: `2020-07-26T18:44:45.869Z`

ISO date time string when this shield information barrier was enabled.
</ParamField>

<ParamField body="enabled_by" type="User (Base)object">
The user that enabled this shield information barrier.
</ParamField>

<ParamField body="enterprise" type="Enterpriseobject">
The `type` and `id` of enterprise this barrier is under.
</ParamField>

<ParamField body="status" type="string">
example: `draft`

Status of the shield information barrier.

Value is one of `draft`,`pending`,`disabled`,`enabled`,`invalid`
</ParamField>

<ParamField body="updated_at" type="string(date-time)">
example: `2020-07-26T18:44:45.869Z`

ISO date time string when this shield information barrier was updated.
</ParamField>

<ParamField body="updated_by" type="User (Base)object">
The user that updated this shield information barrier.
</ParamField>
