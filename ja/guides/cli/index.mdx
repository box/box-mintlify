Box Command Line Interface (CLI) は、ターミナルウィンドウまたはコマンドプロンプトからBox APIにリクエストを行うためのツールです。

## クイックスタート

<Link href="/guides/cli/quick-start">OAuth 2.0を使用したBox CLIの使用を開始する</Link>。

## CLIの設定と使用

* <Link href="/guides/authentication/jwt">サーバー認証</Link>方法を使用するには、<Link href="/guides/cli/cli-docs/jwt-cli">こちら</Link>のガイドを参照してください。
* <Link href="/guides/authentication/client-credentials">CCG認証</Link>方法を使用するには、[こちら][ccg-page]のガイドを参照してください。
* CLIコマンドを使用して一括操作を行うには、[こちら][bulk]のガイドを参照してください。

## サンプルスクリプト

Box CLIの使用自体が役に立ちますが、PowerShellスクリプトと組み合わせるとさらに強力になります。反復タスクの自動化を開始できるように、スクリプトテンプレートの[ライブラリ][scripts]を作成しました。

* <Link href="/guides/cli/scripts/index">Explore sample scripts</Link>
