Boxでウェブリンクを削除するには、そのウェブリンクのIDをAPIに渡す必要があります。

<CodeGroup>

```sh cURL
curl -i -X DELETE "https://api.box.com/2.0/web_links/12345" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```

```typescript Node/TypeScript v10
await client.webLinks.deleteWebLinkById(webLinkId);
```

```python Python v10
client.web_links.delete_web_link_by_id(web_link_id)
```

```cs .NET v10
await client.WebLinks.DeleteWebLinkByIdAsync(webLinkId: webLinkId);
```

```swift Swift v10
try await client.webLinks.deleteWebLinkById(webLinkId: webLinkId)
```

```java Java v10
client.getWebLinks().deleteWebLinkById(weblink.getId())
```

```java Java v5
BoxWebLink webLink = new BoxWebLink(api, id);
webLink.delete();
```

```python Python v4
client.web_link('12345').delete()
print('The web link was deleted!')
```

```cs .NET v6
await client.WebLinksManager.DeleteWebLinkAsync("11111");
```

```javascript Node v4
client.weblinks.delete('11111')
	.then(() => {
		// deletion succeeded — no value returned
	});
```
