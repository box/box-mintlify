ファイルのコンテンツではなく、ファイル自体の情報を取得するには、ファイルの`id`を指定して[`GET /files/:id`](/ja/reference/get-files-id) APIを呼び出します。

```sh cURL
curl -i -X GET "https://api.box.com/2.0/files/12345" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```

## ファイルID

ファイルの`id`を確認するには、ウェブアプリでファイルにアクセスして、URLから`id`をコピーします。たとえば、URLが`https://*.app.box.com/file/123`の場合、`file_id`は`123`です。

## その他のフィールド

ファイルのフィールドをさらに取得するには、必ず`fields`クエリパラメータを渡してください。
