デフォルトでは、検索APIによって**標準**形式の[ファイル](/ja/reference/resources/file)、[フォルダ](/ja/reference/resources/folder)、または[ウェブリンク](/ja/reference/resources/web-link)が返されます。これらの各リソースでは、`fields`クエリパラメータを使用してリクエストできる追加のフィールドがサポートされています。

<CodeGroup>

```curl cURL
curl -i -X GET "https://api.box.com/2.0/search?query=sales&fields=name,tags" \
    -H "Authorization: Bearer <ACCESS_TOKEN>"

```

```java Java
long offsetValue = 0;
long limitValue = 10;

BoxSearch boxSearch = new BoxSearch(api);
BoxSearchParameters searchParams = new BoxSearchParameters();
searchParams.setQuery("sales");

final List<String> fields = new ArrayList<String>();
fields.add("name");
fields.add("tags");
searchParams.setFields(fields)

PartialCollection<BoxItem.Info> searchResults = boxSearch.searchRange(offsetValue, limitValue, searchParams);

```


```csharp .NET
IEnumerable<string> fields = new List<string>() { "name", "tags"};
BoxCollection<BoxItem> results = await client.SearchManager
    .QueryAsync("sales", fields: fields);

```


```python Python
client.search().query("sales", metadata_filters=metadata_search_filters, fields=["name", "tags"])

```

```js Node
client.search.query(
    'sales',
    {
        fields: "name,tags"
    })
    .then(results => {
        // ...
    });

```
</CodeGroup>

<Info>

これらのフィールドの詳細については、[ファイル (詳細)](/ja/reference/resources/file--full)、[フォルダ (詳細)](/ja/reference/resources/folder--full)、および[ウェブリンク (詳細)](/ja/reference/resources/web-link) のリソースを確認してください。

</Info>

<Warning>

`fields`パラメータを使用して、項目の追加情報をクエリで照会すると、これらのフィールドといくつかの**基本**フィールド (`id`、`type`、`name`など) のみが返されます。レスポンスに最初から含まれていたフィールドは、明示的にリクエストする必要があります。

</Warning>
