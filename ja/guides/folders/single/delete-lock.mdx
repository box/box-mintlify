Box内のフォルダに適用されたロックを削除するには、フォルダのロックIDを指定して`DELETE /folder_locks/:id` APIを呼び出します。

<Note>
**フォルダロック**

フォルダロックAPIエンドポイントを使用する際は、アクセスしようとしているフォルダの所有者/共同所有者として認証されている必要があります。

</Note>

<CodeGroup>

```sh cURL
curl -i -X DELETE "https://api.box.com/2.0/folder_locks/93134" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```

```typescript Node/TypeScript v10
await client.folderLocks.deleteFolderLockById(folderLock.id!);
```

```python Python v10
client.folder_locks.delete_folder_lock_by_id(folder_lock.id)
```

```cs .NET v10
await client.FolderLocks.DeleteFolderLockByIdAsync(folderLockId: NullableUtils.Unwrap(folderLock.Id));
```

```swift Swift v10
try await client.folderLocks.deleteFolderLockById(folderLockId: folderLock.id!)
```

```java Java v10
client.getFolderLocks().deleteFolderLockById(folderLock.getId())
```

</CodeGroup>

<Note>
**フォルダのロックIDを見つける方法**

フォルダのロックを削除するには、フォルダのロックIDをAPIに指定する必要があります。フォルダのロックIDは、[フォルダのロックを作成](/ja/guides/folders/single/create-lock)したとき、または特定のフォルダに対する[ロックのリストを取得](/ja/guides/folders/single/get-locks)したときに、レスポンスに示されます。

</Note>
