アプリケーションでは、ファイルをHTML要素としてページに埋め込む場合があります。たとえば、オーディオプレーヤーを使用する場合です。

```html
<audio controls>
  <source src="..." type="audio/mp3">
</audio>
```

この場合、通常の<Link href="/guides/downloads/get-url">ダウンロードURL</Link>を使用しても機能しません。`dl.boxcloud.com`ドメインでは<Link href="/guides/security/cors">クロスオリジンリソース共有</Link>がサポートされていないためです。

その代わり、アプリケーションでは次の形式を使用できます。

```sh
https://api.box.com/2.0/files/[FILE_ID]/content?access_token=[ACCESS_TOKEN]
```

<Warning>
  **CORS**

  これを機能させるために、アプリケーションでは、このファイルをホストするウェブサイトのドメインを<Link href="/guides/security/cors">CORSの設定</Link>で許可されたドメインのリストに追加しておく必要があります。
</Warning>

<Warning>
  **トークンのダウンスコープ**

  この方法を使用すると、アクセストークンがエンドユーザーに公開され、エンドユーザーはこのトークンを使用すると、意図したより多くの操作を行うことができる可能性があります。そのため、状況に応じてこのトークンを<Link href="/guides/authentication/tokens/downscope">ダウンスコープ</Link>することをお勧めします。
</Warning>
