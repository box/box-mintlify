アクセストークンまたは更新トークンを<Link href="/reference/post-oauth2-revoke">`POST /oauth2/revoke`</Link>エンドポイントに送信することにより、アクセストークンをいつでも取り消すことができます。

<CodeGroup>
  ```sh cURL
  curl -i -X POST "https://api.box.com/oauth2/revoke" \
       -H "content-type: application/x-www-form-urlencoded" \
       -d "client_id=[CLIENT_ID]" \
       -d "client_secret=[CLIENT_SECRET]" \
       -d "token=[ACCESS_TOKEN]"
  ```

  ```typescript Node/TypeScript v10
  await auth.revokeTokens();
  // client's tokens have been revoked
  ```

  ```python Python v10
  client.auth.revoke_token()
  ```

  ```cs .NET v10
  await auth.RevokeTokenAsync();
  ```

  ```swift Swift v10
  try await auth.revokeToken()
  ```

  ```java Java v10
  auth.revokeToken();
  // client's tokens have been revoked
  ```

  ```python Python v4
  oauth.revoke()
  ```

  ```javascript Node v4
  client.revokeTokens("<TOKEN>")
  	.then(() => {
  		// the client's access token have been revoked
  	});
  ```
</CodeGroup>

<Info>
  **SDKでの使用方法**

  すべてのBox SDKでは、クライアントに関連付けられた現在のアクセストークンを手動で取り消すことがサポートされています。特定のトークンを取り消すには、最初にそのトークンで新しいSDKを初期化してから、関連する取り消しメソッドを呼び出します。
</Info>
