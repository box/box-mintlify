APIバージョン 2025.0

<Note>
Learn more about [Box SDK versioning strategy](/guides/tooling/sdks/sdk-versioning/).
</Note>

指定したEnterpriseのEnterprise構成。

<ParamField body="id" type="string">
  例 `11446498`

  Enterprise構成の識別子 (EnterpriseのID)。
</ParamField>

<ParamField body="type" type="string">
  例 `enterprise_configuration`

  値は常に`enterprise_configuration`になります。

  次の値に固定: `enterprise_configuration`
</ParamField>

<ParamField body="content_and_sharing" type="Enterprise構成 (コンテンツと共有) object">
  コンテンツと共有カテゴリのEnterprise構成。
</ParamField>

<ParamField body="security" type="Enterprise構成 (セキュリティ) object">
  セキュリティカテゴリのEnterprise構成。
</ParamField>

<ParamField body="shield" type="Enterprise構成 (Shield) object">
  ShieldカテゴリのEnterprise構成。指定したEnterpriseのShieldが無効になっている場合、レスポンスは404になります。
</ParamField>

<ParamField body="user_settings" type="Enterprise構成 (ユーザー設定) object">
  ユーザー設定カテゴリのEnterprise構成。
</ParamField>
