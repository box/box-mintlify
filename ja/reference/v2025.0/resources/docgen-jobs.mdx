title: "Box Doc Genジョブ"
---

APIバージョン 2025.0


標準のパラメータのセットを含むBox Doc Genジョブのリスト。

* [Full](/ja/reference/v2025.0/resources/docgen-jobs--full/)
* [Standard](/ja/reference/v2025.0/resources/docgen-jobs/)

<Info>
このリソースにはAPIの使用時に発生する可能性があるいくつかのバリエーションがあります。

**Base**バリアントに含まれるフィールドは、`fields`クエリパラメータで指定されたフィールドのほかに、そのパラメータを使用しているときに必ず返されるデフォルトのフィールドを表します。
</Info>

<ParamField body="entries" type="Box Doc Genジョブarray">
ジョブのリスト。
</ParamField>

<ParamField body="limit" type="integer(int64)">
例 `1000`

これらのエントリで使用された制限値。許容される最大値を超えていない限り、クエリパラメータの`limit`と等しくなります。最大値はAPIによって異なります。
</ParamField>

<ParamField body="next_marker" type="string">
例 `JV9IRGZmieiBasejOG9yDCRNgd2ymoZIbjsxbJMjIs3kioVii`

次の検索結果ページの開始場所のマーカー。
</ParamField>

<ParamField body="prev_marker" type="string">
例 `JV9IRGZmieiBasejOG9yDCRNgd2ymoZIbjsxbJMjIs3kioVih`

前の検索結果ページの開始場所のマーカー。
</ParamField>
