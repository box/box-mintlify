<ParamField body="external_collab_multi_factor_auth_settings" type="object">
外部コラボレータの多要素認証設定。

<Expandable>
  <ParamField body="is_used" type="boolean">

  指定したEnterpriseに構成が使用されているかどうかを示します。
  </ParamField>

  <ParamField body="value" type="object">
  外部コラボレーションのセキュリティ設定。
  
  <Expandable>
    <ParamField body="allowlist_domains" type="stringarray">
    例: `["例.com"]`
    
    外部コラボレーションが許可されているドメインのリスト。状態が`allowlist`の場合に適用されます。
    </ParamField>
  
    <ParamField body="allowlist_emails" type="stringarray">
    例: `["mail@例.com"]`
    
    外部コラボレーションが許可されているメールアドレスのリスト。状態が`allowlist`の場合に適用されます。
    </ParamField>
  
    <ParamField body="denylist_domains" type="stringarray">
    例: `["例.com"]`
    
    外部コラボレーションが許可されていないドメインのリスト。状態が`denylist`の場合に適用されます。
    </ParamField>
  
    <ParamField body="denylist_emails" type="stringarray">
    例: `["spam@例.com"]`
    
    外部コラボレーションが許可されていないメールアドレスのリスト。状態が`denylist`の場合に適用されます。
    </ParamField>
  
    <ParamField body="factor_type_settings" type="string">
    例: `totp`
    
    外部コラボレータの認証の要素の種類。使用可能な値には`totp`、`any`、`unknown`があります。
    </ParamField>
  
    <ParamField body="scheduled_at" type="string(date-time)">
    例: `2012-12-12T10:53:43-08:00`
    
    スケジュールされている日時。
    </ParamField>
  
    <ParamField body="scheduled_status" type="string">
    例: `inprogress`
    
    外部コラボレーションのセキュリティ設定を適用するためのスケジューリングの状態。使用可能な値には`in_progress`、`scheduled`、`completed`、`failed`、`scheduled_immediate`があります。
    </ParamField>
  
    <ParamField body="state" type="string">
    例: `enabled`
    
    外部コラボレーションのセキュリティ設定の状態。使用可能な値には`enabled`、`disabled`、`allowlist`、`denylist`があります。
    </ParamField>
