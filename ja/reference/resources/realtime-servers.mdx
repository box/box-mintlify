Long pollingに使用できるリアルタイムサーバーのリスト。

<ParamField body="chunk_size" type="integer(int64)">
例 `1`

このレスポンス内の項目の数。
</ParamField>

<ParamField body="entries" type="リアルタイムサーバーarray">
リアルタイムサーバーのリスト。
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "chunk_size": 1,
  "entries": [
    {
      "max_retries": "10",
      "retry_timeout": 610,
      "ttl": "10",
      "type": "realtime_server",
      "url": "http://2.realtime.services.box.net/subscribe?channel=cc807c9c4869ffb1c81a&stream_type=all"
    }
  ]
}
```

</ResponseExample>
