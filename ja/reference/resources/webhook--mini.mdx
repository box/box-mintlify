---
title: "Webhook (Mini)"
---

APIバージョン 2024.0


設定されたWebhookを表します。

* [Standard](/ja/reference/resources/webhook/)
* [Mini](/ja/reference/resources/webhook--mini/)

<Info>
このリソースにはAPIの使用時に発生する可能性があるいくつかのバリエーションがあります。

通常、**Mini**バリアントに含まれるフィールドは、このリソースが別のリソースのレスポンス内にネストされている場合に返されます。
</Info>

<ParamField body="id" type="string">
例 `11446498`

このWebhookの一意の識別子。
</ParamField>

<ParamField body="type" type="string">
例 `webhook`

値は常に`webhook`になります。

次の値に固定: `webhook`
</ParamField>

<ParamField body="target" type="object">
Webhookをトリガーする項目。

<Expandable>
  <ParamField body="id" type="string">
  例 `1231232`
  
  Webhookをトリガーする項目のID。
  </ParamField>

  <ParamField body="type" type="string">
  例 `file`
  
  Webhookをトリガーする項目のタイプ。
  
  次の値のいずれか1つ: `file`,`folder`
  </ParamField>

</Expandable>
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "id": "11446498",
  "type": "webhook",
  "target": {
    "id": "1231232",
    "type": "file"
  }
}
```

</ResponseExample>
