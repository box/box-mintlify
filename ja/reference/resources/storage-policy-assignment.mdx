---
title: "ストレージポリシー割り当て"
---

APIバージョン 2024.0

ユーザーまたは企業へのストレージポリシーの割り当て。

<ParamField body="id" type="string">
例 `ZW50ZXJwcmlzZV8xMjM0NTY3ODkw`

ストレージポリシー割り当ての一意の識別子。
</ParamField>

<ParamField body="type" type="string">
例 `storage_policy_assignment`

値は常に`storage_policy_assignment`になります。

次の値に固定: `storage_policy_assignment`
</ParamField>

<ParamField body="assigned_to" type="object">
そのポリシーが割り当てられる企業またはユーザー。

<Expandable>
  <ParamField body="id" type="string">
  例 `11446498`
  
  このオブジェクトの一意の識別子。
  </ParamField>

  <ParamField body="type" type="string">
  例 `file`
  
  このオブジェクトのタイプ。
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="storage_policy" type="ストレージポリシー (Mini)object">
割り当てられたストレージポリシー。
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "id": "ZW50ZXJwcmlzZV8xMjM0NTY3ODkw",
  "type": "storage_policy_assignment",
  "assigned_to": {
    "id": "11446498",
    "type": "file"
  },
  "storage_policy": {
    "id": "11446498",
    "type": "storage_policy"
  }
}
```

</ResponseExample>
