---
title: "メタデータフィールドのフィルタ (浮動小数点の範囲)"
---

APIバージョン 2024.0

検索結果のフィルタに使用するテンプレートの`float`フィールドを指定して、照合する値の範囲を指定します。

<ParamField body="gt" type="number">
例 `100000`

メタデータフィールド値の下限 (その値を含む) を指定します。検索クエリがこのテンプレートに一致するには、フィールドの値をこの値より大きくする (`gt`) かこの値と等しくする必要があります。
</ParamField>

<ParamField body="lt" type="number">
例 `200000`

メタデータフィールド値の上限 (その値を含む) を指定します。検索クエリがこのテンプレートに一致するには、フィールドの値をこの値より小さくする (`lt`) かこの値と等しくする必要があります。
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "gt": 100000,
  "lt": 200000
}
```

</ResponseExample>
