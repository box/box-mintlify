---
title: "タイムラインスキルカード"
---

APIバージョン 2024.0

<Note>
このリソースは、バージョン**2024.0**のエンドポイントで使用されています。
詳細については、
**[Box APIのバージョン管理](/ja/guides/api-calls/api-versioning-strategy/)**を参照してください。

「[Box SDKのバージョニング戦略](/ja/guides/tooling/sdks/sdk-versioning/)について詳しく学ぶ。」
</Note>

タイムラインに画像のリストを配置するBox Skillメタデータカード。

<ParamField body="type" type="string">
例 `skill_card`

値は常に`skill_card`になります。

次の値に固定: `skill_card`
</ParamField>

<ParamField body="created_at" type="string(date-time)">
例 `2018-04-13T13:53:23-07:00`

このカードの作成日時 (省略可)。
</ParamField>

<ParamField body="duration" type="integer">
例 `1000`

タイムラインの総持続時間 (秒)。
</ParamField>

<ParamField body="entries" type="object[]">
タイムライン上のエントリのリスト。

<Expandable>
  <ParamField body="appears" type="object[]">
  この項目がタイムラインに表示される日時を表すタイムスタンプのリストを定義します。
  
  <Expandable>
    <ParamField body="end" type="integer">
    例 `20`
    
    エントリがタイムラインに表示されなくなる時間 (秒)。
    </ParamField>
  
    <ParamField body="start" type="integer">
    例 `1`
    
    エントリがタイムライン上に表示され始める時間 (秒)。
    </ParamField>
  
  </Expandable>
  </ParamField>

  <ParamField body="image_url" type="string">
  例 `https://example.com/image1.jpg`
  
  タイムラインに表示されるエントリを示す画像。この画像のURLはすべてのエントリに必要です。
  
  この画像は項目 (顔など) のリストに表示されます。また、画像をクリックすると、このエントリの期間中にそのエントリが表示される位置がユーザーに示されます。
  </ParamField>

  <ParamField body="text" type="string">
  例 `John`
  
  エントリのテキスト。これは、たとえば、ビデオで検出された人物の名前など、タイムラインに配置される項目の表示名になります。
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="invocation" type="object">
このサービスの呼び出し。メタデータを適用したサービスのインスタンスを追跡するために使用されます。

<Expandable>
  <ParamField body="id" type="string">
  例 `image-recognition-service-123`
  
  このメタデータを適用したサービスのインスタンスを表すカスタム識別子。たとえば、`image-recognition-service`が複数のノードで動作する場合、このフィールドを使用すると、メタデータの適用に使用されたノードのIDを識別できます。
  </ParamField>

  <ParamField body="type" type="string">
  例 `skill_invocation`
  
  値は常に`skill_invocation`になります。
  
  次の値に固定: `skill_invocation`
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="skill" type="object">
このメタデータを適用したサービス。

<Expandable>
  <ParamField body="id" type="string">
  例 `image-recognition-service`
  
  このメタデータを適用したサービスを表すカスタム識別子。
  </ParamField>

  <ParamField body="type" type="string">
  例 `service`
  
  値は常に`service`になります。
  
  次の値に固定: `service`
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="skill_card_title" type="object">
カードのタイトル。

<Expandable>
  <ParamField body="code" type="string">
  例 `Faces`
  
  タイトルの識別子 (省略可)。
  </ParamField>

  <ParamField body="message" type="string">
  例 `Faces`
  
  UI上で表示する実際のタイトル。
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="skill_card_type" type="string">
例 `timeline`

値は常に`timeline`になります。

次の値に固定: `timeline`
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "type": "skill_card",
  "created_at": "2018-04-13T13:53:23-07:00",

      "appears": [

          "end": 20,

        }
      ],

  ],

  },

    "id": "image-recognition-service",

  },
  "skill_card_title": {
    "code": "Faces",
    "message": "Faces"
  },
  "skill_card_type": "timeline"
}
```

</ResponseExample>
