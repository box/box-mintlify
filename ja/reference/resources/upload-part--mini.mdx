アップロードセッションチャンクのBase版の表示。

* [Standard](/ja/reference/resources/upload-part/)
* [Mini](/ja/reference/resources/upload-part--mini/)

<Info>
このリソースにはAPIの使用時に発生する可能性があるいくつかのバリエーションがあります。

通常、**Mini**バリアントに含まれるフィールドは、このリソースが別のリソースのレスポンス内にネストされている場合に返されます。
</Info>

<ParamField body="offset" type="integer(int64)">
例 `16777216`

ファイル内のチャンクのオフセット (バイト数)。ファイル内のチャンクの位置の下限です。
</ParamField>

<ParamField body="part_id" type="string">
例 `6F2D3486`

チャンクの一意のID。
</ParamField>

<ParamField body="size" type="integer(int64)">
例 `3222784`

チャンクのサイズ (バイト単位)。
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "offset": 16777216,
  "part_id": "6F2D3486",
  "size": 3222784
}
```

</ResponseExample>
