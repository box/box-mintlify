---
title: "Slack統合マッピングリクエストを作成"
---

APIバージョン 2024.0


Slack統合マッピングオブジェクトを作成するリクエスト。

<ParamField body="box_item" type="object">
Slackタイプの統合マッピングのBox項目オブジェクトのスキーマ。

<Expandable>
  <ParamField body="id" type="string">
  例 `1234567891`
  
  (`type`で参照されるタイプの) マッピングされた項目のID。
  </ParamField>

  <ParamField body="type" type="string">
  例 `folder`
  
  `id`で参照されるマッピングされた項目のタイプ。
  
  次の値に固定: `folder`
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="options" type="object">
Slackタイプの統合マッピングオプションオブジェクトのスキーマ。

<Expandable>
  <ParamField body="is_access_management_disabled" type="boolean">
  例 `true`
  
  基になるBox項目に対するチャンネルメンバーのアクセスを自動で管理する必要があるかどうかを示します。チャンネルのタイプによっては、アクセスがコラボレーションまたは共有リンクの作成により管理されます。
  </ParamField>

</Expandable>
</ParamField>

<ParamField body="partner_item" type="object">
Slackタイプの統合マッピングのマッピングされた項目オブジェクトのスキーマ。

Box for Slackがオーガナイゼーションレベルとワークスペースレベルのどちらでインストールされているかに応じて、`slack_org_id`**または**`slack_workspace_id`の**いずれか**を指定します。両方のパラメータを同時に使用しないでください。

<Expandable>
  <ParamField body="id" type="string">
  例 `C12378991223`
  
  (`type`で参照されるタイプの) マッピングされた項目のID。
  </ParamField>

  <ParamField body="type" type="string">
  例 `channel`
  
  `id`で参照されるマッピングされた項目のタイプ。
  
  次の値に固定: `channel`
  </ParamField>

  <ParamField body="slack_org_id" type="string">
  例 `E1234567`
  
  項目が関連付けられているSlackオーガナイゼーションのID。このパラメータは、オーガナイゼーションレベルでBox for Slackがインストールされている場合に使用します。`slack_workspace_id`を同時に使用しないでください。
  </ParamField>

  <ParamField body="slack_workspace_id" type="string">
  例 `T12352314`
  
  項目が関連付けられているSlackワークスペースのID。このパラメータは、ワークスペースレベルでBox for Slackがインストールされている場合に使用します。`slack_org_id`を同時に使用しないでください。
  </ParamField>

</Expandable>
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "box_item": {
    "id": "1234567891",
    "type": "folder"
  },
  "options": {
    "is_access_management_disabled": true
  },
  "partner_item": {
    "id": "C12378991223",
    "type": "channel",
    "slack_org_id": "E1234567",
    "slack_workspace_id": "T12352314"
  }
}
```

</ResponseExample>
