クエリの処理に使用されるAIエージェント。

<ParamField body="id" type="string">
例 `14031`

エージェントのID。これは、カスタムエージェントの場合は数値ID (例: `14031`)、組み込みのエージェントの場合は一意の識別子 (例: [抽出エージェント (強化)](/ja/guides/box-ai/ai-tutorials/extract-metadata-structured//#enhanced-extract-agent) の場合は`enhanced_extract_agent`) になります。
</ParamField>

<ParamField body="type" type="string">
例 `ai_agent_id`

クエリの処理に使用されるAIエージェントのタイプ。

次の値に固定: `ai_agent_id`
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "id": "14031",
  "type": "ai_agent_id"
}
```

</ResponseExample>
