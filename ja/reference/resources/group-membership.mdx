title: "グループメンバーシップ"
---

APIバージョン 2024.0

<Note>
このリソースは、バージョン**2024.0**のエンドポイントで使用されています。
詳細については、
**[Box APIのバージョン管理](/ja/guides/api-calls/api-versioning-strategy/)**を参照してください。

「[Box SDKのバージョニング戦略](/ja/guides/tooling/sdks/sdk-versioning/)について詳しく学ぶ。」
</Note>

メンバーシップは、ユーザーがグループに属していることを示すために使用されます。

<ParamField body="id" type="string">
例 `11446498`

このグループメンバーシップの一意の識別子。
</ParamField>

<ParamField body="type" type="string">
例 `group_membership`

値は常に`group_membership`になります。

次の値に固定: `group_membership`
</ParamField>

<ParamField body="created_at" type="string(date-time)">
例 `2012-12-12T10:53:43-08:00`

このメンバーシップが作成された日時。
</ParamField>

<ParamField body="group" type="グループ (Mini)object">
メンバーシップが適用されるグループ。
</ParamField>

<ParamField body="modified_at" type="string(date-time)">
例 `2012-12-12T10:53:43-08:00`

このメンバーシップが最後に変更された日時。
</ParamField>

<ParamField body="role" type="string">
例 `member`

グループ内のユーザーのロール。

次の値のいずれか1つ: `member`,`admin`
</ParamField>

<ParamField body="user" type="ユーザー (Mini)object">
メンバーシップが適用されるユーザー。
</ParamField>

<ResponseExample>

```json レスポンスの例
{

    "type": "group",
    "group_type": "managed_group",
    "name": "Support"

  }
}
```

</ResponseExample>
