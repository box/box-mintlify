---
title: "ウェブリンク (Base)"
---

APIバージョン 2024.0

ウェブリンクとは、URLを指すオブジェクトです。これらのオブジェクトはBoxウェブアプリケーション内ではブックマークとも呼ばれます。

ウェブリンクオブジェクトはファイルオブジェクトと同様に扱われ、通常のファイルに適用されるアクションの大部分もサポートしています。

* [Standard](/ja/reference/resources/web-link/)
* [Mini](/ja/reference/resources/web-link--mini/)
* [Base](/ja/reference/resources/web-link--base/)

<Info>
このリソースにはAPIの使用時に発生する可能性があるいくつかのバリエーションがあります。

**Base**バリアントに含まれるフィールドは、`fields`クエリパラメータで指定されたフィールドのほかに、そのパラメータを使用しているときに必ず返されるデフォルトのフィールドを表します。
</Info>

<ParamField body="id" type="string">
例 `11446498`

このウェブリンクの一意の識別子。
</ParamField>

<ParamField body="type" type="string">
例 `web_link`

値は常に`web_link`になります。

次の値に固定: `web_link`
</ParamField>

<ParamField body="etag" type="string">
例 `1`

このウェブリンクのエンティティタグ。`If-Match`ヘッダーで使用します。
</ParamField>

<ResponseExample>

```json レスポンスの例
{
  "id": "11446498",
  "type": "web_link",
  "etag": "1"
}
```

</ResponseExample>
