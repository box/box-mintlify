デバイスピンのリスト。

<ParamField body="entries" type="Device Pinnerarray">
デバイスピンのリスト。
</ParamField>

<ParamField body="limit" type="integer(int64)">
例 `200`

これらのエントリで使用された制限値。許容される最大値を超えていない限り、クエリパラメータの`limit`と等しくなります。
</ParamField>

<ParamField body="next_marker" type="integer(int64)">
例 `3000`

次の検索結果ページの開始場所のマーカー。
</ParamField>

<ParamField body="order" type="object[]">
項目が返される順序。

<Expandable>
  <ParamField body="by" type="string">
  例 `id`
  
  並べ替えに使用するフィールド。
  
  次の値に固定: `id`
  </ParamField>

  <ParamField body="direction" type="string">
  例 `asc`
  
  並べ替えの方向 (昇順または降順)。
  
  次の値のいずれか1つ: `asc`,`desc`
  </ParamField>

</Expandable>
</ParamField>

{
