To get information about an assigned task, call the
[`GET /tasks/:task_id`](/reference/get-tasks-id) API with the
`id` of the task.

<CodeGroup>
```sh cURL
curl -i -X GET "https://api.box.com/2.0/tasks/12345" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```

```typescript Node/TypeScript v10
await client.tasks.getTaskById(task.id!);
```

```python Python v10
client.tasks.get_task_by_id(task.id)
```

```csharp .NET v10
await client.Tasks.GetTaskByIdAsync(taskId: NullableUtils.Unwrap(task.Id));
```

```swift Swift v10
try await client.tasks.getTaskById(taskId: task.id!)
```

```java Java v10
client.getTasks().getTaskById(task.getId())
```

```java Java v5
BoxTask task = new BoxTask(api, "id");
BoxTask.Info info = task.getInfo();
```

```py Python v4
task = client.task(task_id='12345').get()
print(f'Task ID is {task.id} and the type is {task.type}')
```

```csharp .NET v6
BoxTask task = await client.TasksManager.GetTaskAsync("11111");
```
