To get information about an assigned task, call the
<Link href="/reference/get-task-assignments-id">`GET /tasks/:task_id/assignments`</Link> API with the
`id` of the task assignment.

<CodeGroup>
```sh cURL
curl -i -X GET "https://api.box.com/2.0/task_assignments/12345" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```
