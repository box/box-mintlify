You can use Box Python SDK to call Box APIs in a Python project.

<Card href="https://github.com/box/box-python-sdk" arrow title="Learn more about Python SDK on GitHub"/>

## Installation

To install the Python SDK run the following command from your terminal
window or command prompt using [pip][pip].

```shell
pip install boxsdk
```

## JWT application

When working with a Box App that uses server-side <Link href="/guides/authentication/jwt">JWT authentication</Link>, install the following additional module:

```shell
pip install "boxsdk[jwt]"
```

[pip]: https://pypi.org/project/pip/
