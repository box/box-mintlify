<Link href="https://getpostman.com">Postman</Link> is a tool that lets you build and test HTTP
requests in an easy-to-use
interface without configuring a full development environment. The Box Postman
Collection is a set of preconfigured requests that make it possible to get
started with the Box API from Postman without having to manually configure the
requests.

<Frame center>
  ![Postman](./postman.png)
</Frame>

## Overview

This guide will take you through the following steps.

1. <Link href="/guides/tooling/postman/quick-start/install-postman">Install Postman</Link> on your device.
2. <Link href="/guides/tooling/postman/quick-start/configure-box-app">Configure a Box App</Link> so that Postman can access the Box API.
3. <Link href="/guides/tooling/postman/quick-start/log-in-to-box">Log in</Link> to the Box App to get the right API credentials.
4. <Link href="/guides/tooling/postman/quick-start/load-postman-collection">Fork Box's Postman collection</Link> into Postman.
5. <Link href="/guides/tooling/postman/quick-start/make-api-call">Make a first API call</Link> to the Box API with Postman.
