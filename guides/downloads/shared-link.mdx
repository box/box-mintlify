To download the file for a <Link href="/guides/shared-links">Shared Link</Link>, first <Link href="/guides/shared-links/find-for-item">determine the
file</Link> for the link.

<Note>
You cannot use the shared link to download folders. <Link href="/guides/downloads/zip-archive">Create and download
the ZIP archive</Link> instead.
</Note>

Once the file ID has been determined, the file can be downloaded by passing the
`BoxAPI` header to the API.
