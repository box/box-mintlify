Deleting a domain from the list of allowed collaboration domains will remove the
ability for collaborations to be created between your enterprise and users in
that domain.

To remove a domain from the allowed list, supply the ID of the list entry to
the delete request. This ID is returned when <Link href="/guides/collaborations/allowed-domains/create">allowing a new domain</Link>
or <Link href="/guides/collaborations/allowed-domains/list">listing the allowed domains in the enterprise</Link>;

<CodeGroup>

```sh cURL
curl -i -X DELETE "https://api.box.com/2.0/collaboration_whitelist_entries/213123" \
     -H "authorization: Bearer <ACCESS_TOKEN>"
```
