This guide describes how to restore content that you archived by mistake.

## Restore file or folder from archive

Use the <Link href="/reference/put-files-id">`PUT /files/:id`</Link> API endpoint to restore a file from an archive, or the <Link href="/reference/put-folders-id">`PUT /folders/:id`</Link> endpoint to restore a folder from an archive.
The `id` parameter is the ID of the file/folder you want to restore from an archive.
To specify the destination, use the `parent.id` <Link href="/reference/put-folders-id/#param-parent-id">parameter</Link> in the request body.
This is the ID of a folder (can be owned by any user) where you want to restore the file/folder.

To restore a file/folder to user's root folder, use `0` as the `parent.id` value. Additionally, pass the ID of the user in the
`parent.user_id` <Link href="/reference/put-folders-id/#param-parent-user_id">parameter</Link> in the request body.
