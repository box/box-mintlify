To add content to an archive, you need to create it first. If you
have not done this yet, use the <Link href="/reference/v2025.0/post-archives/">Create Archive</Link> endpoint.

## Add file or folder to archive

Use the <Link href="/reference/put-files-id">`PUT /files/:id`</Link> API endpoint to add a file to an archive, or the <Link href="/reference/put-folders-id">`PUT /folders/:id`</Link> endpoint to add a folder.
The `id` parameter is the ID of the file/folder you want to add to the archive.
To specify the destination, use the `parent.id` <Link href="/reference/put-folders-id/#param-parent-id">parameter</Link> in the request body.
This can be either an ID of an archive or the ID of a folder that is inside an archive.

[Update parent id of file]: /reference/put-files-id/#param-parent-id
