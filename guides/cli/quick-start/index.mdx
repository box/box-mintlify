The Box CLI is a user-friendly command line tool that allows both technical and 
non-technical users to leverage the Box API to perform routine or bulk actions.
There is no need to write any code, as these actions are executed through a set
of [commands][commands].

## Overview

This guide will take you through the following steps.

1. <Link href="/guides/cli/quick-start/create-oauth-app">Create and configure</Link> a Box application
2. <Link href="/guides/cli/quick-start/install-and-configure">Install and configure</Link> the CLI
3. <Link href="/guides/cli/quick-start/build-commands-help">Run commands</Link> with the CLI
4. Use <Link href="/guides/cli/quick-start/options-and-bulk-commands">options and bulk commands</Link>
5. Explore automation using <Link href="/guides/cli/quick-start/powershell-script-templates">Powershell scripts with the CLI</Link>
6. <Link href="/guides/cli/quick-start/next-steps">Next steps</Link>
