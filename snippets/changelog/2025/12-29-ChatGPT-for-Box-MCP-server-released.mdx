## ChatGPT for remote Box MCP server released
We’ve added the ability to [integrate ChatGPT with the Remote Box MCP Server][gpt-remote-box-mcp]. This allows ChatGPT to securely access Box content through a custom MCP connector using OAuth and the Model Context Protocol (MCP).

## What's included in this launch

* The ability to add the remote Box MCP server in ChatGPT.
* [Step‑by‑step instructions][gpt-remote-box-mcp] for authorizing ChatGPT access to Box.

## Developer impact

This change enables developers to connect ChatGPT with their Box environment through the Remote MCP Server, allowing secure query and interaction with Box data from within ChatGPT workflows.
