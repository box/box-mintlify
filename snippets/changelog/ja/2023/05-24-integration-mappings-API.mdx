## 統合マッピングAPI

  [統合マッピング][1]を使用すると、パートナーアプリからのコンテンツがBoxのどこに保存されるかを管理できます。

現在、この機能では[SlackのコンテンツレイヤーとしてBoxを使用する][2]場合のみサポートします。

## サポート情報

問題がある場合やさらにガイドが必要な場合は、必要なサポートについて、Boxの[Developer Forum][3]に英語でリクエストを投稿してください。
