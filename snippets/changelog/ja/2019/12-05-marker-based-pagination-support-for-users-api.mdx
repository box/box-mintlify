<Update label={<span>Users APIでマーカーベースのページネーションがサポート対象に</span>} description="2019-12-05" tags={["新機能", "API"]}>
  [Users API](e://get_users)で[マーカーベースのページネーション](g://api-calls/pagination/marker-based)がサポートされるようになりました。マーカーベースのページネーションとは、通常のオフセットベースのページネーションに代わるもので、多数のユーザーを有する会社のユーザーを取得する代替手段を提供します。
