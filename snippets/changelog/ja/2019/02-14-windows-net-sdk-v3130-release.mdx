## Windows .NET SDK v3.13.0のリリース
  * `BoxCollaboration`オブジェクトに`.InviteEmail`プロパティを追加。このプロパティは、保留中のコラボレーションで招待されたユーザーのメールアドレスを表示します。
* `BoxUser`オブジェクトに`.Timezone`、`.IsExternalCollabRestricted`、`.Tags`、および`.Hostname`プロパティを追加
